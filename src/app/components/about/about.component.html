Sobre nosotros

Con CryptoPass podrás hacer uso de nuestro estupendo conversor de códigos hash, que hace uso de los 3 tipos más comunes de algoritmos hash: MD5, SHA-256 y SHA-512

¿Quieres convertir cualquier texto a uno de estos formatos? CryptoPass es para ti.

Servicios

Gestor de contraseñas

¿Alguna vez has soñado con no tener que recordar todas las contraseñas que usas diariamente? Si es así, usar esta herramienta te vendrá genial.

Solo tendrás que registrarte y empezar a almacenar tus contraseñas, codificadas haciendo uso de algoritmos de cifrado para aportar mayor seguridad.

Conversor MD5

El algoritmo MD5 se trata de un algoritmo de reducción criptográfico de 128 bits ampliamente usado.

Con nuestro sencillo conversor, podrás codificar cualquier texto que introduzcas haciendo uso de este algoritmo.

Conversor SHA-256

El algoritmo SHA-256 forma parte del conjunto de funciones hash SHA-2, diseñadas por la Agencia de Seguridad Nacional de los EEUU (NSA). Concretamente, este algoritmo produce un resultado de 256 bits.

Con nuestro sencillo conversor, podrás codificar cualquier texto que introduzcas haciendo uso de este algoritmo.

Conversor SHA-512

El algoritmo SHA-512 forma parte del conjunto de funciones hash SHA-2, diseñadas por la Agencia de Seguridad Nacional de los EEUU (NSA). Concretamente, este algoritmo produce un resultado de 512 bits, lo que lo convierte en el más robusto de esta familia de algoritmos.

Con nuestro sencillo conversor, podrás codificar cualquier texto que introduzcas haciendo uso de este algoritmo.
